# 关于 VEP

<待编辑>

VEP（Vix Enhancement Proposal）是 Vix 语言的改进提案流程，受 Python PEP 启发。

## 提案类型

- **Standard** — 语言特性、标准库等核心变更
- **Process** — 围绕 Vix 开发流程的变更
- **Informational** — 设计指南、信息说明

## 提案状态

| 状态 | 说明 |
|------|------|
| Draft | 草稿，讨论中 |
| Accepted | 已接受，待实现 |
| Implemented | 已实现 |
| Rejected | 已拒绝 |
| Withdrawn | 已撤回 |
| Final | 最终定稿 |
