# 关于 VEP

VEP（Vix Enhancement Proposal）是 Vix 语言的改进提案流程，受 Python PEP 启发。

## 提案类型

| 类型 | 说明 |
|------|------|
| **标准** | 语言特性、标准库等核心变更 |
| **流程** | 围绕 Vix 开发流程的变更 |
| **信息** | 设计指南、信息说明 |

## 提案状态

| 状态 | 说明 |
|------|------|
| **草稿** | 正在编写和讨论中 |
| **已接受** | 核心团队认可方向，待实现 |
| **已实现** | 已完成开发 |
| **已拒绝** | 经讨论后被否决 |
| **已撤回** | 提交者主动撤回 |
| **终稿** | 已实现并稳定 |

详细流程说明见 [VEP 1 — VEP 目的与流程]({{ '/veps/vep-0001/' | relative_url }})。
